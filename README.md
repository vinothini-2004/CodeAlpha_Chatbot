# FAQ Chatbot v2 — 100% Working (No Flask)

## Files
- server.py     ← main file to run
- faqs.json     ← FAQ data
- requirements.txt

## Run Steps

### Step 1 — Install only ONE library
```
pip install nltk
```

### Step 2 — Run the server
```
python server.py
```

### Step 3 — Open browser
```
http://localhost:8000
```

That's it! DO NOT open index.html directly in browser.
